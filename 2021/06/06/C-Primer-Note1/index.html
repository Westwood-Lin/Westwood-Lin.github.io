C++ Primer Part1

​

第一部分就第一章，没什么内容；

1.简介了iostream库，包括2个类 istream和ostream，cin是istream类的对象实例，cout、cerr、clog是ostream类的对象实例；

2.>>和<<运算符本质上是语法糖，因为istream和ostream是定义在头文件中，对<<和>>的重载必须是全局函数；

​ 函数的参数是一个绑定stream对象的引用，另一边是个变量或常量；同样返回一个stream对象的引用；

3.命名空间、作用域运算符 ::

4.循环输入的最简单方式：

​ int value;

​ while(cin>>value){ do sth; }

5.第一章涉及概念

​ argument / parameter. function. return type. signature. body. name.

​ block. *buffer. built-in type. library type. (character) string literal. comment.

​ condition. curly brace. edit-compile-debug. expression. statement. data structure.

​ EOF. initialize. *uninitialized varible. source file. header. #include

​ class. namespace. iostream. cin cout clog cerr. *manipulator. method. member function.

std. standard input. standard output. standard error.

​ Operator: >, >=, <, <=, ==, !=, ++, –, +=, =

​ >>, <<, . , :: , ()

*buffer :为解决IO设备和CPU的速度差，在两者间增加了一个buffer。buffer是一块存储区域。IO设备将输入或输出的数据保存在缓冲区中；可以显式地刷新缓冲区，以便将正确地让程序从console读取输入/产生到屏幕的输出。

​ 例如：一个常见的错误：在 cin之后用getline

​ { int intA; string strB;

cin>>intA; getline(cin,strB);

} 中输入 2 [endl] hello [EOF].

​ 会发现strB不能读入”hello”. 这是因为缓冲区同样会记录回车的 [endl]，getline（）从缓冲区读入，读到的第一个字符就是endl，直接就返回了。
